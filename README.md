# 🚀 Port Ninja

A stealth port scanner written in Python for penetration testers.

---

## 🎯 Features

- TCP port scanning with banner grabbing
- (Optional) UDP scanning support
- Multithreaded scanning
- JSON report output

---

```bash
python3 main.py -t 192.168.1.1 --tcp --ports 1-1000
